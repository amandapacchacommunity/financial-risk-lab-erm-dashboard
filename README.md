# Financial Risk Lab: ERM Dashboard (2025)

## Overview
This project is a synthetic enterprise risk management and financial dashboard designed for a mission-driven organization with approximately $10M in annual revenue.

It demonstrates how financial data, organizational risk, donor exposure, and executive reporting can be integrated into a single decision-support model.

## What This Demonstrates
- Integration of financial risk and enterprise risk management
- Board-level reporting for mission-driven organizations
- Donor attrition and margin analysis
- Executive-ready financial sustainability insights

## Key Features
- Financial performance indicators
- Risk exposure modeling
- Donor attrition risk view
- Board summary metrics
- Scenario-oriented decision support

## Example Use Cases
This type of model could support:
- Nonprofit financial planning
- Board reporting
- ERM committee review
- Budget risk assessment
- Long-term sustainability planning

## Data Note
All data is synthetic and created for demonstration purposes only.

## About This Work
This project shows how finance, risk, and governance can be connected to support clearer decisions and long-term institutional stability.
